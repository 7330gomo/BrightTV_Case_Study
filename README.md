# BrightTV_Case-Study

# Project Overview

BrightTV’s CEO aims to increase the company’s subscription base during the current financial year. To support this objective, this case study focuses on analyzing user engagement and subscription trends to provide actionable insights for the Customer Value Management (CVM) team.

# Objectives

Understand user behavior across different channels and content types.

Identify low-engagement periods or underperforming content.

Recommend strategies to increase subscription uptake and improve retention.

Provide data-driven insights to support CVM decision-making.


# Dataset Description

The dataset contains information on BrightTV users, including:

User demographics: age, gender, province, and race.

Viewing behavior: channels watched, duration, and frequency.

Engagement metrics: total users, weekly watches, monthly engagement, and content preferences.


# Analysis Goals

User vs. Channel Engagement – Analyze which channels and content types attract the most viewers.

User Demographics vs. Viewing Patterns – Understand which age groups, genders, or provinces have higher engagement.

Content Optimization – Identify opportunities to create or promote content to increase consumption.

Subscription Growth Opportunities – Highlight periods, channels, or demographics with potential for subscription campaigns.


# Key Deliverables

Pivot tables and charts summarizing user engagement trends.

Insights on high- and low-performing channels and content.

Recommendations for CVM strategies to boost subscriptions.

Actionable visualizations for management presentations.
